## Long Description

**ClaudeWatch — Community Claude Usage Tracker**

Anthropic doesn't publish Claude's usage limits or tell you when they change. ClaudeWatch fills that gap by pooling anonymous usage metadata from thousands of users into a public, real-time dashboard.

**What ClaudeWatch does:**

- Estimates your token usage per message (computed locally — no text is transmitted)
- Tracks your daily and all-time usage stats right in the popup
- Shows your usage percentage and reset timer from the Claude settings page
- Detects when you hit a rate or usage limit
- Reports response times and tokens-per-second (a proxy for server-side speed)
- Pools this data anonymously with the community to reveal global trends

**What ClaudeWatch never does:**

- Read, store, or transmit any message content or AI responses
- Ask you to log in or create an account
- Access any page outside claude.ai
- Sell your data to anyone

**The community dashboard (free, public) shows:**

- Rate limit events per day — detect when limits tighten
- Avg tokens per response over time — track Claude's output patterns
- Response speed trends — spot infrastructure changes
- Usage distribution by tier (Free vs Pro)
- Model distribution across the community
- A community board for observations (280 chars, anonymous, no links)

**Privacy by design:**

Your anonymous ID is a randomly generated UUID stored only on your device. A one-way hash of browser characteristics is used server-side only to detect duplicate submissions — the raw values are never sent. No email, no account, no tracking across sites.

Open source. Fully auditable. No hidden data collection.

## Data Collection Disclosure (for Store listing form)

This extension collects the following data, as required by Chrome Web Store policy:

**Personal communications:** No.

**Financial and payment information:** No.

**Authentication information:** No.

**Personal communications:** No.

**Location:** No.

**Web history:** No (only claude.ai activity is observed).

**User activity:** Yes — interaction metadata on claude.ai (estimated token counts, response times, usage percentages). No content is collected.

**Website content:** No.

**Personally identifiable information:** No. All data is associated with a randomly generated anonymous ID.

Data is transmitted to a Supabase backend for community aggregation. It is never sold. The full privacy policy is available at https://dashboard-five-gamma-82.vercel.app/privacy
